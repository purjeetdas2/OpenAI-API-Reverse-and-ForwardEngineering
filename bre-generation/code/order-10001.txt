1. COBOL Program (ORDERPGM):
```
IDENTIFICATION DIVISION.
PROGRAM-ID. ORDERPGM.
DATA DIVISION.
WORKING-STORAGE SECTION.
EXEC SQL INCLUDE SQLCA END-EXEC.
LINKAGE SECTION.
01 ORDER-ID PIC X(10).
PROCEDURE DIVISION.
EXEC SQL DECLARE C1 CURSOR FOR
SELECT order_id
FROM order_table
WHERE order_id = :ORDER-ID
END-EXEC.
```

2. Subroutine (VALIDATESUB):
```
IDENTIFICATION DIVISION.
PROGRAM-ID. VALIDATESUB.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 SQL-CA.
  05 FILLER PIC S9(8) COMP.
  05 SQLCODE PIC S9(9) COMP.
01 ORDER-ID PIC X(10).
LINKAGE SECTION.
01 SUB-ORDER-ID PIC X(10).
PROCEDURE DIVISION USING SUB-ORDER-ID.
MOVE SUB-ORDER-ID TO ORDER-ID.
EXEC SQL OPEN C1 END-EXEC.
IF SQLCODE = 0
   PERFORM VALID-ORDER
ELSE
   PERFORM INVALID-ORDER
END-IF.

VALID-ORDER.
DISPLAY 'Order ID is valid'.
EXEC SQL CLOSE C1 END-EXEC.

INVALID-ORDER.
DISPLAY 'Order ID is invalid'.
GOBACK.
```
